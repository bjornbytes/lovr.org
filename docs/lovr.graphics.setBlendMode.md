lovr.graphics.setBlendMode
===

Sets the blend mode.

    lovr.graphics.getColor(blend, alphaBlend)

### Arguments

- `BlendMode blend` The blend mode used to blend colors.
- `BlendAlphaMode alphaBlend (alphamultiply)` The blend mode used to blend alpha values.

### Returns

Nothing
